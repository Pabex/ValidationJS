ValidatorJS Docs

createValidator($form, validationTrigger, options)
Parmámetros
  • $form: objeto jquery que representa el formulario.
  • validationTrigger: puede ser:
    • VALIDATE_ON_SUBMIT: las valdaciones se disparan al enviar el formuario.
    • VALIDATE_ON_BLUR: las valdaciones se disparan al abandonar cada campo del formulario.
    • VALIDATE_ON_CHANGE: las valdaciones se disparan al cambiar cada campo del formulario.
  • options: JSON con propiedades:
    • validValidation: función callback al comprobar una validación válida en un campo.
    • invalidValidation: función callback al comprobar una validación inválida en un campo.
    • validField: función callback al comprobar que todas las validaciones en un campo son válidas.
    • invalidField: función callback al comprobar que al menos una de las validaciones en un campo es inválida.
    • validForm: functión callback al comprobar que todo el formulario es válido.
    • invalidForm: functión callback al comprobar que al menos un campo en el formulario es inválido.
Funciones callback
  • validValidation(options:{ event: "Evento", validation: "Objeto validation", field: "$Campo", message: "Mensaje de validación" })
  • invalidValidation(options:{ event: "Evento", validation: "Objeto validation", field: "$Campo", message: "Mensaje de validación" })
  • validField({ event: "Evento", field: "$Campo", messages: "Array de mensajes" })
  • invalidField({ event: "Evento", field: "$Campo", messages: "Array de mensajes" })
  • validForm({ event: "Evento", validator: "Objeto validator", form: "$Formulario" })
  • invalidForm({ event: "Evento", validator: "Objeto validator", form: "$Formulario" })
